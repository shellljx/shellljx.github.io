发表于 2015-02-21   |   分类于 技术杂烩   |  

网上关于Ghost主题MarkDown代码块高亮的教程很多，下面这种方法我感觉是最简单的。

使用Prism是简单并且轻量级的方法，效果如下:

Prism非常容易安装，只要下载下来自定义的css和js文件，然后放到你的主题相应位置就可以运行了。

###install Prism

  1. 进入 http://prismjs.com/download.html
  2. 选择一些设置，比如你要什么主题、你想支持什么语言等等.
  3. 滚动到页面的最下面，下载你自定义好的Prism.js和Prism.css文件.
  4. Copy这两个文件到你的服务器.
  5. Copy Prism.js到ghost/content/themes/主题名字/assets/js目录下.
  6. Copy Prism.css到ghost/content/themes/主题名字/assets/css目录下.
  7. 修改/ghost/content/themes/主题名字/default.hbs文件.

###注意事项

  1. 在Markdown代码块的顶部确保使用了language-blahblah，否则Prism将不起作用。
  2. Prism也会对其他主题起作用，但是有些主题已经使用了自己的代码高亮，你应该先去掉默认高亮效果或者替代他们。
