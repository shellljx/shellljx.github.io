Are you ok?

Aug 4, 2017

我不是针对谁，我只是想说国产手机厂商深度定制的 ROM 都是垃圾！垃圾！垃圾！

今天调试程序需要修改 host 通过 Charles 代理的方式来访问测试服，登录接口使用的是 https 需要在手机上安装证书才能访问，发现 MiUI 恶心人的一个地方就是竟然无法安装证书，一直提示无法找到证书文件的错误，浪费了我半个小时搞这个破问题，然后换了一个其他品牌的手机顺利的安装上了证书。
MiUI 不仅不能安装证书，而且系统设置入口极深，自定义的各种系统设置看的眼花缭乱，更可气的是开启 USB 调试需要登录小米账号，are you ok ？
MiUI 号称“针对中国用户使用习惯开发的Android Rom” ，很难想象在小米的眼里，中国用户的使用习惯是什么。我只看到了一个复杂臃肿的手机系统，我只看到了一个口口声声说为了用户的使用安全，实则为了增加自己对系统的控制权而处处设限的一个 ROM。
